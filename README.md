# Tacoma Encampment Cleanup Map

This repo serves a static Leaflet map using:

- `data/ES_Encampment_Cleaning_Tracking.geojson`
- `data/SeeClickFix_Requests.geojson`

## Run locally with Docker

```bash
docker compose up --build
```

Open:

- http://localhost:8080

Stop:

```bash
docker compose down
```
